
# FlagFrenzy
FlagFrenzy consists of cybersecurity challenges through a capture the flag game for users to put their skills to the test with a series of different cybersecurity scenarios.
It is beginner friendly, as it's been designed for non-advanced Linux users. The game is played through a website where users login/signup and try out the challenges that are displayed. 
Leaderboard ranking are present, consistently displaying the best solution/fastest solution for each challenge.

Requirements:
- A laptop that's able to access the internet.
- A linux terminal.
- A stable internet connection.
- Some type of knowledge on basic linux commands.

Key Features : 
- User Login/SignUp.
- Real-Time Leaderboard rankings.
- Security features. 

Structure: 
Front-End : React.js
Back-End: Node.js
Databases: MySQL
